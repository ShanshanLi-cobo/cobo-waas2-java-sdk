

# TokenizationSOLTransferFeeExtensionParams

Transfer Fee Extension configuration for SPL Token 2022. This extension enables automatic fee collection on every token transfer. The fee is calculated as a percentage of the transfer amount in basis points. 

## Properties

| Name | Type | Description | Notes |
|------------ | ------------- | ------------- | -------------|
|**transferFeeBasisPoints** | **Integer** | The transfer fee rate in basis points (1/100 of a percent). For example, 100 basis points equals 1% fee. Maximum value is 10000 (100%).  |  |
